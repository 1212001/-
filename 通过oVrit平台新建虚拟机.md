## git 相关命令的使用

1、下载项目到本地(克隆)：

​	   git clone 项目地址

2、创建分支（master为主分支）

​	   git branch 分支名

3、查看当前分支（前面带*的代表当前分支）

​		git branch

4、查看所有分支

​		git branch -a

5、移动到分支

​		git chekout 分支名

6、提交

​		git add 文件名

​		git commit -m "输入提交的信息"

​		git push origin 分支名
